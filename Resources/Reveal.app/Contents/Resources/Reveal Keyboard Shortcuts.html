iOS Simulator

When the iOS Simulator is the foreground active application pressing the following keyboard shortcuts will launch Reveal.app (if it is not already running) and initiate the specified command. For the command to succeed the application running in the Simulator must have the Reveal library loaded/linked and the Reveal server started.

Action Effect
⇧⌘E Refresh View Hierarchy

If necessary this keyboard shortcut can be changed in the Keyboard System Preferences Panel.
